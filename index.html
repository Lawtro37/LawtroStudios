Solar View

An educational solar sim programmed using p5.js

"Solar View" is one of my notable projects, and it holds a special place in my heart. I developed this educational solar simulator using the versatile p5.js library. The main goal behind this project was to make learning about the solar system an engaging and interactive experience for students.

Solar View allows users to explore and interact with a virtual representation of our solar system. You can learn about the planets, their orbits, and other celestial objects in a fun and immersive way. The project was initially created for a former primary school, where it received a positive response from both students and educators.

To explore "Solar View" and embark on a journey through the cosmos, simply follow this [link](./solar-view) or watch the video below. It's an excellent resource for students, teachers, and anyone curious about our celestial neighborhood.

Custom Built 3D Renderer

A custom 3D renderer for JavaScript

I created a custom 3D renderer for JavaScript that allows for the creation of custom models using JSON files. This renderer provides the capability to toggle outlines and change the size of the shape. I am currently working on expanding its functionality to support multiple shapes and possibly developing a library with dynamic camera features and enhanced scripting capabilities.

This project has been an exciting journey into the world of 3D graphics, where I've gained insights into the inner workings of rendering engines and learned to optimize performance for complex models. The renderer is built using modern web technologies, making it accessible for a wide range of applications.

I look forward to further enhancing this project and exploring the possibilities of interactive 3D experiences on the web. Stay tuned for updates and additional features as I continue to push the boundaries of what can be achieved with JavaScript and 3D rendering.

Cellular AI

A Genetic Cellular AI

"Cellular" is an exciting project that combines cellular particals with genetic AI techniques. It's designed to simulate complex cellular life, and I'm currently working on evolving it into a powerful and flexible tool for simulatig cellular AI.

The project is inspired by the fascinating dynamics of cells and the adaptability of genetic algorithms. Cellular will provide a versatile framework to study patterns and behaviors in various fields, from biology to physics and beyond. Genetic algorithms, on the other hand, allow for the optimization and evolution of these cells to survive for as long as posible.

In "Cellular," I've combined these concepts to create a platform where cellular structures can evolve and adapt to changing environments. This project has the potential for a wide range of applications, including the study of emergent behavior in complex systems, the optimization of cellular structures for specific tasks, and the exploration of creative and artistic possibilities.
